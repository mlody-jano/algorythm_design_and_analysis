# Algorythm Desing and Analysis

## Introduction

Repository containing projects done during consecutive classes of **_Algorythm Design and Analysis_** course, which was responsible for providing students with a grasp understanding of how algorythms work, how to properly write optimal algorythms, and much more.

## Tech stack
- Language - **C++**

## First Project - Sorting algorythms (mergesort, quicksort, introsort)

First project mentioned in the following repository is an attempt of implementing 3 particular sorting algorythms, that being:
- mergesort()
- quicksort()
- introsort()

Algorythms were implemented without the usage of the **STL** library. After implementation, every algorythm was tested on a specific test case, with a selected number of records being sorted.
